How It Works
Bulk Assignment Process:
  1. Select multiple users from the list
  2. Choose menu items to grant access to
  3. Optionally assign a role to all users
  4. Select specific permissions (View, Create, Edit, Delete, Approve)
  5. Review the preview and submit
Benefits:
  • Save time by assigning access to multiple users at once
  • Ensure consistent permissions across user groups
  • Easy role-based access management
  • Bulk updates for existing access assignments
